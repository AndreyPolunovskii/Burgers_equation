
                                           Burgers_equation_MKR

    Данная программа Burgers_equation_MKR предназначена для расчета решений уравнения Бюргерса-Хопфа 
методом разложения в степенные ряды по переменной времени t, и дескритизации методом конечных разностей по пространственной переменной.

Требования:
 Для работы программы требуется установленные gnuplot и библиотека openMP.  


Структура программы : 

  Интерфейс программы Burgers_equation_MKR представляет собой систему папок , расположенных в основной папке Burgers_equation_MKR . В ней находятся папки sourse, documentation, (создается в процессе расчета) project, experiments и файлы RUN и PARAMETERS.h . 

  Папка sourse содержит файлы иходного кода, необходимого для работы программы.
  
  Папка documentation содержит документацию по программе.
 
  Папка experiments содержит сохраненные по усмотрению пользователя вычисления, сделанные в данной программе.

  Папка project содержит результаты данного расчета (см. ниже).

 Файл RUN представляет собой исполняемый файл, требуемый для запуска программы.
 
 Файл PARAMETERS.h представляет собой заголовочный файл с параметрами решения (см. ниже) . 
  
  

Запуск программы:

Для запуска программы откройте терминал в папке Burgers_equation_MKR и выполните команду

./RUN


Сразу начнется расчет , который может продолжаться в зависимости от установленных параметров
  долго или быстро.


По окончании расчета появится информация о его завершении, и будут созданы графики.

 Далее по завершении выполнения программы будет перезаписана папка project в папке burgers_equation_MKR.
В данной папке project будут находиться четыре папки : data, bin, graphics, log, и один файл PARAMETERS.h .
 
В папке data находятся файлы output_data_animate_2.txt и output_data_point_funct_2.txt . 
    В файле output_data_point_funct_2.txt выписываются в каждом i-ом (начиная со второго) столбике значения i-ой расчетной точки в различные моменты расчетного времени. В первом столбике выписаны  значения данных моментов времени.
    В файле output_data_animate_2.txt выписываются несколько блоков, каждый их которых по два столбика. В первом стоблике выписаны расчетные точки пространственного отрезка, на котором строится решение. Во втором столбике выписаны сами значения решения в данных точках отрезка. Каждый блок соответствует определенному моменту времени (Отсчет времени начинается с первого блока, поэтому первый блок фактически соответсвует начальному условию). 
   
В папке bin находится бинарный исполняемый файл, скомпилированый при запуске программы.

В папке log находится logfile.log , в котором  выписывется информация о выполнении расчета .

В папке graphics находятся три файла animate.gif , graph_moment.gif и X.eps . 
    animate.gif представляет собой анимацию решения , graph_moment.gif - график решения, построенный в фиксированный момент времени, заданный в файле PARAMETERS.h (см. ниже). 
    В файле X.eps построены графики нескольких первых расчетных точек, количество которых определяется в файле PARAMETERS.h (см. ниже).

 Файл PARAMETERS.h в папке project представлет собой просто копию файла PARAMETERS.h из папки burgers_equation_MKR , в котором находятся параметры расчета (см. ниже).
